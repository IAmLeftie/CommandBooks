# Command Books
Adds book items that can be configured by admins to perform commands on use. Books are configurable individually via chat commands, given that the user is an Operator.

Command Books cannot be crafted, and must be obtained in Creative mode.
<br>
## Commands
* /commandbook commands list - Lists the held command book's commands
* /commandbook commands add [command\] - Adds the specified command to the command book.
* /commandbook commands remove [index\] - Removes the command at the specified index.
* /commandbook onetimeuse [true|false\] - Sets whether the book is destroyed after one use.
* /commandbook elevated [true|false\] - Sets whether the book should run the command with the highest permissions available.
* /commandbook secret [true|false\] - Sets whether the book's commands should be hidden from the player.
